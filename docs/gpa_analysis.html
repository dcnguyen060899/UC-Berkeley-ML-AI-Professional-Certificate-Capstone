Academic Growth Analysis Dashboard

This project visualizes academic growth across various phases of student performance using GPA data. It leverages React and Recharts to present insights on performance trends and phase-wise stability.

Key Features

  • Phase-wise GPA trend visualization
  • Volatility analysis for academic stability
  • Interactive tooltips with contextual data
  • Reference lines for significant milestones
  • Clear depiction of performance improvement

Interactive Dashboard

Benefits

  • Comprehensive academic performance tracking
  • Identification of critical transition phases
  • Insight into factors driving performance stability
  • Visual representation of long-term improvement
  • Enhanced decision-making for academic advisors
